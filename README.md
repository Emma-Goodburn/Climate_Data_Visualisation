# Running the website

- Navigate to the root folder for this project  
- Run `npm install` to ensure all node packages are installed  
- Run `npm run server` to start the NodeJS server  
- Run `npm run preview` in a new terminal to run the currently built version of the code  
- If this fails first run `npm run build`, and then repeat the previous step  
- Open [localhost:4173](localhost:4173) in a browser to view the website
